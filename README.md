## Installation
- Clone the repository:
```
  git clone https://github.com/SWP-APALARY/APALARY-Frontend.git
  cd APALARY-Frontend
```
- Using yarn to install and start app:
```
  yarn install
  yarn dev
  Or
  npm install
  npm run dev
```
- Enjoy yourself!
